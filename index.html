What is the difference between rowspan and colspan?
Rowspan allows you to merge rows in a table while colspan allows you to merge columns.
What is the difference between cells spacing and padding?
Cells spacing is the space between two adjacent cells. Cell padding is the space between the cell content and the border that surrounds it.
How do you define table borders using CSS?
To define table borders, you use the border property. To control border widths, you use the border-width property. The border-width property can include between one and four values (px, solid line, etc).
